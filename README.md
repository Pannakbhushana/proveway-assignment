# BOGO Offer

## Project Overview

A simple **BOGO Offer** UI built using **HTML, CSS, and JavaScript**.
The project displays multiple offer boxes with variants, discount badges, and a responsive layout suitable for desktop and mobile screens.

## Features

* Clean and minimal UI
* Responsive design for desktop, tablet, and mobile screens
* Interactive offer selection with radio buttons
* Active offer highlights with detailed information
* “Add to Cart” button aligned with the selected offer
* Popular offer badge
* Footer showing delivery info and total price
* Fully modular CSS structure (`base.css`, `layout.css`, `components.css`, `main.css`)
